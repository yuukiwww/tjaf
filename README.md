# tjaf

太鼓ウェブの曲を管理します

## デバッグの開始

依存関係をインストールします

```bash
pip install -r requirements.txt
```

サーバーを起動します

```bash
SONGS_DIR=~/taiko-web/public/songs fastapi dev
```
